Autonomous Ambulance System

Coursework (Team Project - Fall 2024)

Embedded Architecture

Collaborating with a team of engineers, I helped architect an embedded navigation system integrating sensor data with motor actuation for autonomous obstacle avoidance. My contributions focused on control logic and sensor fusion:

  • Embedded Sensing: Integrated Ultrasonic sensors (HC-SR04) for real-time distance measurements.
  • Actuation & Control: Implemented PWM motor commands via an L298N H-Bridge.
  • Obstacle Avoidance Logic: Built a decision-making state machine handling halts, pivots, and rerouting.
  • Validation (Testing): Iteratively tuned thresholds and behavior to prevent corner traps and false positives.
