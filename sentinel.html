Sentinel: Target Tracking & Intercept

Computer Vision Target Tracking & Intercept System

Winter 2025 — Personal Project
Status: In Progress — Building Over Winter Break

Engineering Cycle

Control loops are useless if you do not know what you are looking at. Sentinel is a C++ computer vision pipeline that uses OpenCV to detect and track moving targets (e.g., drones or debris), estimates their 3D trajectory with a Kalman Filter, and computes an intercept vector in real-time.

This is an ongoing Winter 2025 build. The core architecture, math, and testing plan are defined; implementation and iteration will happen over winter break, with progress logged as detection, tracking, and intercept modules come online.

  • Detection & Preprocessing: Implement motion-based and learned detectors (background subtraction / YOLO-style models) on incoming frames, including denoising, thresholding, and contour extraction for small, fast-moving targets.
  • Tracking & State Estimation: Use a Kalman Filter to estimate target position and velocity in 3D, fuse multiple observations over time, and maintain a stable track through occlusions and noise.
  • Intercept Solution: Formulate the relative motion problem and compute a firing solution (lead angle / intercept vector) that accounts for target velocity, system response time, and actuator limits.
  • Real-Time C++ Pipeline: Build a low-latency execution loop in C++ / OpenCV with profiling hooks, ensuring deterministic timing and a clean API that can plug into a future targeting gimbal or UAV controller.
